настройка mysql сервера в netbeans
http://netbeans.org/kb/docs/ide/mysql.html

1. выбираем базы данных слева -> правой кнопкой -> зарегистрировать сервер mysql
2. появится "Сервер mysql в localhost"
3. правой кнопкой на нем -> connect
4. появятся все базы данных -> выбираем нужную -> connect
5. под списком баз данных появятся оранжевые иконки с jdbc:mysql://localhost:3306/gxsblogs?zeroDateTimeBehavior=convertToNull [root на Схема по умолчанию]
6. нажать -> откроется список баз данных с + - в них таблицы по каждой
7. правой кнопкой на таблицу -> просмотр данных -> откроется файл, где можно править данные таблицы. можно также писать mysql команды и нажимать выполнить (бочонок со стрелкой в левом верхнем углу)
